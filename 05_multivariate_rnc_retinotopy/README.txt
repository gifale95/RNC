To analyze the same controlling images as in the RNC paper, you need to load the stats file https://github.com/gifale95/RNC/blob/main/05_multivariate_rnc_retinotopy/stats.py and add it to the results directory "../relational_neural_control/multivariate_rnc/stats/cv-0/imageset-nsd/V1-hV4/stats.py".

The NSD V4 ventral and dorsal manual delineations are found in "https://github.com/gifale95/RNC/blob/main/05_multivariate_rnc_retinotopy/delineate_nsd_v4_dorsal_ventral/nsd_v4_delineations", where both surface and volume labels are available. Before running "02_retinotopy_effect.py", the volume labels "https://github.com/gifale95/RNC/blob/main/05_multivariate_rnc_retinotopy/delineate_nsd_v4_dorsal_ventral/nsd_v4_delineations/volume_labels/nsddata/ppdata/subj0*/func1pt8mm/roi/V4*.nii.gz" need to be added to the corresponding Natural Scenes Dataset directory "../natural-scenes-dataset/nsddata/ppdata/subj0*/func1pt8mm/roi/V4*.nii.gz".
